Average SFR Value
(Jan 2015 - Jun 2020)

This heat map displays the average property value of a Single Family Residence (SFR) in the state of Texas by county, and is filtered to give a semi-annual look at how the recorded property values for the state changed.

This heatmap was made using Tableau with data sourced using Zillow's API.
