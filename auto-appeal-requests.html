Appeal Form Automations

Project for: Altus Group

Problem

There are a large number of properties to be appealed and the company must first notify the asseesor that we are going to represent the clients. In spite of how the forms to do so are in a standard format, that standard is different for each assessor and appeal type. Filling out these forms is a time consuming process and only becomes worse each year.

Solution

Utilized Power Query in Excel to extract, transform, and load the data from the company's database to create a CSV to be used with the Evermap AutoMailMergeâ„¢ software. The Evermap software was used to fill out the PDF form and create a new PDF for each property record. Then a Visual Basic macro was run to combine the resulting PDF with its associated supporting files. This was made to be handled by the use of a Windows batch (.bat) file to allow for non-technical employees to accomplish the task with the ease of clicking the file.

Result

The team was able to complete their appeal forms for the applicable properties more quickly than in prior years, and are able to more easily transfer the process to new employees.
